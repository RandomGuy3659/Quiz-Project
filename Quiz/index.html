There are three questions, with three answers each. Only one is considered "correct" for the purposes of this quiz.

Question Object has a String for question (Question text), and three answer objects.
Answers have a string for answer (Answer.Text) and boolean for if it's the correct answer (verity).

On play, reset value of correct answers, then go to the first question.
On correct answer, value of correct answers +1


Question (current)/(chosen max): (insert question here) in large font, (insert answers here.)
clicking an answer checks the chosen answer for a true in the verity check.
If correct: stuff temporarily turns green, you got it! Is displayed, and the value is updated.
Else: stuff goes red, you got it wrong is displayed, continue.
